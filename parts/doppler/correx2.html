Échantillonnage du signal et spectre en fréquence¶

Dans l'nonc, on parle d'un microphone dans l'habitacle enregistrant le son avec une frquence d'chantillonnage $f_s$ de 2048 Hz : la valeur de la tension cre par l'lectroaimant prsent dans le microphone sous l'effet des variations de la pression acoustique est enregistre toutes les ${1\over 2048}\approx 0,49\,\mathrm{ms}$. C'est ce que fait en ralit tous les instruments de mesure numrique, en particulier un microphone. Pour une dure d'enregistrement donne, la frquence d'chantillonnage dtermine en effet le nombre de valeurs enregistres en l'espace d'une seconde, et est donc le paramtre selon lequel on peut juger de la fidlit d'un enregistrement.

On dfinit la fonction correspondant au signal $s(t)$ mis par la sirne, et on en construit un chantillon de 2 secondes.

def signal_func(t):
    """Fonction signal s(t)"""
    harm = [435,580] # frquences des harmoniques
    amp = [1.,1/3] # amplitudes des harmoniques
    phas = [0.6,0] # phases associes
    return sum(
        am*np.cos(2*np.pi*f*t - ph) 
        for (am,f,ph) in zip(amp,harm,phas))

def echanti():
    """Construit l'chantillon du signal s(t)"""
    global fs
    fs = 2048 # frquence d'chantillonnage
    global dt
    dt = 1/fs # pas d'chantillonnage
    global N
    N = 4096 # nombre d'chantillons
    global times
    times = np.linspace(0,N*dt,N+1) # points t_k d'chantillonnement de s(t) ;
    # on chantillonne donc sur N*dt = 2.0 secondes
    global signal_temp
    signal_temp = signal_func(times) # chantillons s_k = s(t_k)
echanti()

Action de l'effet Doppler sur un signal composé¶

On sait quantifier comment l'effet Doppler agit sur un son pur : il modifie sa frquence $f$ selon la formule donne. Mais pour un signal compos comme ici, nous n'avons aucun moyen de procder de faon directe.

L'ide est la suivante : on utilise le principe de dcomposition de Fourier. En effet $s(t)$ est la somme de sons purs dont les caractristiques telles que la frquence sont donnes par son spectre.

Afin de savoir ce qu'il va faire au signal $s(t)$ mis par la sirne, il suffit donc de dterminer comment sera modifie chaque frquence intervenant dans sa dcomposition.

D'aprs le spectre donn, le signal $s(t)$ contient deux composantes harmoniques principales, de frquences

$$ f_1 = 435\,\mathrm{Hz}\quad\text{et}\quad f_2 = 580\,\mathrm{Hz}. $$

La vitesse du vhicule de police est $v=70\,\mathrm{km.h^{-1}} = 19,4\,\mathrm{m.s^{-1}}\approx 0.057c$ : on va viter d'utiliser la formule approche. Le produit scalaire vaut $\vec v\cdot\vec u= \|\vec v\|\cos(30) = 19,4\times \frac{\sqrt 3}{2}$ (puisque $\|\vec u\| = 1$).

Les nouvelles frquences sont donc, en utilisant la formule (2):

\begin{equation*} \begin{array}{ll} f_1' &= \dfrac{c}{c-\vec v\cdot\vec u}f_1 \\ &= \dfrac{340}{340-19,4\times\cos(30)}\times 435 = 458\,\mathrm{Hz} \end{array}\quad\text{et}\quad f_2' = 610\,\mathrm{Hz}. \end{equation*}

def doppled():
    freq_new = [458,610] # frquences obtenues par effet Doppler
    u = list(zip(freqs,signal_fr))
    amps_c = [c[1] for c in u if c[0] in [435,580]]  
    # rcupre les amplitudes des harmoniques initiaux
    amps, phas = (np.abs(amps_c),np.angle(amps_c)) 
    global Nr ; Nr = N
    global tia; tia = np.linspace(0,2,Nr+1)
    global fia; fia = np.linspace(0,fs/2,Nr//2+1)
    def four(fr):
        """Construit le spectre"""
        if fr in freq_new:
            i = freq_new.index(fr)
            return amps[i]*np.exp(2*np.pi*fr*1j-phas[i]*1j)
        else: 
            return 0
    
    global signal_rec_f; signal_rec_f = [four(fr) for fr in fia] # Spectre
    global signal_rec
    signal_rec = np.fft.irfft(signal_rec_f) # Signal reconstitu
